> https://code.visualstudio.com/api

<img src="./snipping.png" style="outline:1px solid black"/>

## 开发

开发阶段，只需要按下`F5`即可。

开发完毕后，准备打包的话，需要安装打包命令：

```
npm install -g vsce
```

然后执行命令：

```
vsce package
```
